Advanced Topics
=========================

These pages preserve the useful advanced material from the older Apptainer
documentation. They are not required for the beginner workflow, but they are
helpful for specialized applications such as Jupyter, RStudio, MATLAB, R,
Nextflow, MCMICRO, and advanced image security.

Before running long or interactive applications, request an appropriate LSF
session. Do not run heavy applications directly on login nodes.
