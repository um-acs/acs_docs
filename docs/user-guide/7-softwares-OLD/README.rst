Software and Modules
====================

This section covers software and environment workflows for IDSC advanced
computing systems. Use these pages for modules, Python, R, Conda, JupyterHub,
RStudio, MATLAB, GPU frameworks, workflow tools, and specialized applications.

.. note::

   Use this section as the main software guide. System-specific details should
   appear inside the relevant page when the workflow depends on architecture,
   queue names, module versions, service URLs, GPU hardware, or package
   availability.

.. caution::

   Module names, software versions, service URLs, and package availability can
   change. Always check the system where the job will run using commands such as
   ``module avail`` and ``module spider``.
